提供泛型转换特性 相对于OkHttp的Callback新增三个回调函数: onSuccessonComplete 这三个函数都运行在主线程上

Parameters

lifecycle

将Activity/Fragment/LifecycleOwner等作为参数传递会在其生命周期结束时自动取消网络请求. 避免内存泄漏

lifeEvent

指定自动取消的生命周期. 默认为Lifecycle.Event.ON_DESTROY
